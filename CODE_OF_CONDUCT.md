# Code of Conduct

ArchePy adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its Code of Conduct.

By participating in this project — as a contributor, maintainer, issue
reporter, or any other role — you agree to abide by its terms.

## Reporting

To report unacceptable behavior, email the maintainer (see the address listed
in [`pyproject.toml`](pyproject.toml) under `[project.authors]`). Reports will
be handled confidentially and reviewed as quickly as possible.

For the full text of the standards, scope, enforcement guidelines, and
attribution, see the canonical document linked above.
